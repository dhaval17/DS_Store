## Python script to view contents of .DS_Store file

### Usage 
`pip install -r requirements.txt`
`python dsstore.py -p /.DS_Store -t Iloc`

## Requirements

### ds_store :
`pip install ds_store`

Types supported 

- Iloc
- bwsp
- lsvp
- lsvP
- icvp
